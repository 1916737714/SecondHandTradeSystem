# Android二手交易系统

#### 介绍
这是一个基与安卓的二手交易系统 使用技术KOLIN + COMPOSE 完成

#### 软件架构
本软件使用MVVM架构
#### 安装教程

1. 安卓androidstudio
2.  打开本项目，等待gradle构建完成，gradle第一次加载需要非常多的时间
